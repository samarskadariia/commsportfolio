# Daria Samarska — Portfolio

Static personal portfolio site.

## Publish on GitHub Pages
1. Create a new repository and upload the **contents of this folder** to the repo root
   (`index.html`, `support.js`, `assets/`, `uploads/`, `.nojekyll`).
2. Repository → **Settings → Pages** → Source: *Deploy from a branch* → Branch: `main` / `/ (root)`.
3. The site goes live at `https://<username>.github.io/<repo>/` within a minute or two.

The `.nojekyll` file must stay — it lets GitHub serve every asset as-is.
